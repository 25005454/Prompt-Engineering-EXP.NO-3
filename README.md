EX.NO:3 Exploring the different types of prompts- Straightforward Prompts - Tabular Format Prompting - Missing Word Prompting - Preceding Question Prompting.

Aim: To write the prompts for the following prompt types for 2 different scenarios of Each type.

1.Straightforward Prompts

2.Tabular Format Prompting

3.Preceding Question Prompting

4.Missing Word Prompting

Explanation - Any one use case from Unit 5 or any 2 own use cases with the unit 2 Prompt type

Procedure:

Write 2 different prompts for different use cases, refer the following Examples

Example:

Straightforward Prompts:

• "Define photosynthesis in one sentence."

"List three advantages of electric vehicles."

Tabular Format Prompting:

• "Compare and contrast AC and DC current in a table."

• "Provide a table listing five programming languages, their paradigms, and one use case each."

Preceding Question Prompting:

• "Why is climate change a global concern? Explain how greenhouse gases contribute to global warming."

• "How do vaccines work? Describe the process of immunization in simple terms."

Missing Word Prompting:

• "The capital of France is ____."

• "In photosynthesis, plants absorb sunlight to produce ____."

OUTPUT:
1. Straightforward Prompts
Definition:

Straightforward prompts are simple and direct instructions given to the AI. These prompts clearly state what information is needed without additional context.

Characteristics:
Simple and direct
Easy to understand
Produces quick responses
Mostly used for definitions, lists, and explanations
Scenario 1:
Use Case:

Education

Prompt:

“Explain the importance of renewable energy in simple words.”

Expected Response:

Renewable energy helps reduce pollution and provides clean energy from natural resources like sunlight, wind, and water.

Scenario 2:
Use Case:

Technology

Prompt:

“List five benefits of learning programming.”

Expected Response:
Improves problem-solving skills
Creates career opportunities
Helps build software and apps
Enhances logical thinking
Supports automation and innovation

2. Tabular Format Prompting
Scenario 1:

Prompt:
“Compare online learning and classroom learning in a table format.
| Feature     | Online Learning | Classroom Learning |
| ----------- | --------------- | ------------------ |
| Flexibility | High            | Limited            |
| Interaction | Virtual         | Face-to-face       |
| Cost        | Less            | More               |

Scenario 2:

Prompt:
“Provide a table showing different types of computer networks and their uses.”
| Network Type | Full Form                 | Use                     |
| ------------ | ------------------------- | ----------------------- |
| LAN          | Local Area Network        | Used in schools/offices |
| MAN          | Metropolitan Area Network | Used within a city      |
| WAN          | Wide Area Network         | Used across countries   |

3. Preceding Question Prompting
Definition:

Preceding question prompting first asks a related question before requesting the main explanation. This helps provide context and improves detailed responses.

Characteristics:
Provides context before the main question
Encourages detailed answers
Improves clarity
Useful in analytical and descriptive tasks
Scenario 1:
Use Case:

Cybersecurity

Prompt:

“Why is cybersecurity important? Explain different methods used to protect data.”

Expected Response:

Cybersecurity is important because it protects personal and organizational data from hackers and cyberattacks. Methods used include firewalls, encryption, passwords, antivirus software, and multi-factor authentication.

Scenario 2:
Use Case:

Healthcare Technology

Prompt:

“How does artificial intelligence help in healthcare? Describe its applications with examples.”

Expected Response:

Artificial intelligence helps healthcare by improving diagnosis, patient monitoring, and treatment planning. Examples include AI-based scanning systems, virtual health assistants, and robotic surgeries.

4. Missing Word Prompting
Definition:

Missing word prompting provides incomplete sentences where the AI or user fills in the missing word. This type is commonly used in quizzes, learning exercises, and language training.

Characteristics:
Interactive learning method
Tests knowledge and memory
Simple and engaging
Used in educational applications
Scenario 1:
Use Case:

Science

Prompt:

“The process of converting water into vapor is called ____.”

Expected Response:

Evaporation

Scenario 2:
Use Case:

Computer Fundamentals

Prompt:

“A computer processes data using the ____ unit.”

Expected Response:

Central Processing Unit (CPU)


RESULT: Thus,the experiment for exploring different type of prompt is completed successfully.
